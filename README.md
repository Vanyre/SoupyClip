# 🍲 SoupyClip

> **清汤寡水 · 只为过审 · 极简主义自动化混剪工作站**
>
> *Automated Video Synthesis Tool for Mac (Apple Silicon)*

## 📖 简介 (Introduction)

**SoupyClip** 是一款专为 Mac (M芯片) 用户打造的高性能视频批量混剪工具。

它的核心理念是 **“对冲不过审”**。通过自动化的剪辑逻辑、随机化的视觉微调和硬件加速引擎，它能将原本枯燥的素材库瞬间转化为成百上千条 **“指纹唯一”** 的短视频，极大提高短视频矩阵的过审率和分发效率。

告别繁琐的手动剪辑，让机器为你打工。🚀

## 🧠 核心工作流与算法逻辑 (How it Works)

SoupyClip 并非简单的视频拼接器，它内置了一套完整的 **非线性编辑自动化流水线 (Automated NLE Pipeline)**。

以下是单条视频生成的全过程解析：

### 1. 智能剧本构建 (Script Generation)

- **随机参数**：系统首先随机设定成品的 **总时长**（17s/18s/19s/20s）和 **片段数量**（6-10段）。
- **时隙分配**：通过算法将总时长“打碎”成若干个随机长度的时间槽（例如：`3.2s` + `1.5s` + `...`），确保节奏自然，非线性。

### 2. 动态素材匹配 (Dynamic Matching)

- **指纹扫描**：脚本会预先扫描所有素材的元数据（时长、分辨率）。
- **智能选角**：对于每一个时间槽（如需 `3.2s`），算法会自动在素材库中寻找一个 **时长 > 3.2s** 的视频源，防止黑屏或报错。
- **随机截取**：在选定素材的有效时长内，随机选取一个 `Start Time` 进行截取，确保每次使用同一素材的片段都不同。

### 3. 单次编码流水线 (Single-Pass Pipeline) 🔥 *Core Tech*

这是 v12.0 内核的核心优势。为了保证画质和速度，我们将所有画面处理操作合并在 **一次编码** 中完成：

> **输入源** $\rightarrow$ **硬件解码** $\rightarrow$ **[ 滤镜链处理 ]** $\rightarrow$ **硬件编码** $\rightarrow$ **中间切片**

**[ 滤镜链处理 ] 包含：**

1. **AI 视觉去重**：对亮度 ($\pm 2\%$)、对比度/饱和度 ($\pm 2\%$)、锐化程度 ($1.0\sim1.05$) 进行**独立随机扰动**。这意味着每条视频的每一帧像素直方图都是独一无二的。
2. **标准化**：强制缩放并填充至 `1080x1920` (竖屏)，自适应各种尺寸素材。
3. **图层合成**：直接在显存中将 `.png` 水印图层“焊”在视频流上，无需二次渲染。

### 4. 极速流式封装 (Stream Copy)

- **无损拼接**：将生成的中间切片通过 `Concat` 协议无缝粘合。
- **音频混流**：随机选取背景音乐，根据视频长度自动裁剪 (`-shortest`)。
- **瞬时导出**：最终合成阶段仅进行音频编码，视频流直接采用 **Copy 模式** 封装，实现秒级导出，且画质零损耗。

## ✨ 核心功能 (Features)

- **⚡️ M芯片原生加速**：深度集成 Apple `VideoToolbox` 硬件编码器，利用 GPU 极速渲染，不发热、不卡顿。
- **🧠 智能项目导入**：只需选择产品主文件夹（如 `HQ`），算法自动根据文件指纹（`.mp4`, `.png`, `.mp3`）识别视频、音频和贴图目录，无需手动配置。
- **🛡️ AI 视觉去重**：像素级画面微调，对抗平台查重算法。
- **🧹 零垃圾残留**：内置 `atexit` 自毁程序，任务结束自动清理所有临时文件，保持电脑洁净。
- **🎨 Notion 风格 UI**：全中文界面，极简黑白配色，带有清晰的进度日志和交互反馈。

## 🛠️ 环境部署指南 (Installation)

如果你是一台全新的 Mac 电脑，请按照以下步骤配置环境。整个过程只需 5 分钟。

### 第一步：安装 Homebrew (如果已安装可跳过)

打开 **终端 (Terminal)**，复制并运行以下命令：

```
/bin/bash -c "$(curl -fsSL [https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh](https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh))"
```

### 第二步：安装核心引擎 FFmpeg

SoupyClip 的底层依赖于强大的 FFmpeg 进行多媒体处理。

```
brew install ffmpeg
```

*验证安装：输入 `ffmpeg -version`，看到版本信息即成功。*

### 第三步：安装 Python 图形库依赖

Mac 自带的 Python 可能不包含图形界面支持 (Tkinter)，推荐使用 Homebrew 安装完整的 Python 环境。

```
brew install python-tk@3.10
```

### 第四步：安装打包工具 (可选)

如果你想修改源码并重新打包成 APP：

```
pip3 install pyinstaller
```

## 🚀 如何使用 (Usage)

### 1. 准备素材库

建议按以下结构整理你的文件夹（命名随意，工具会自动识别内容）：

```
📂 我的产品_HQ/
  ├── 📂 视频素材/    (存放 .mp4, .mov 等无声或有声片段，建议时长 >1s)
  ├── 📂 顶部贴图/    (存放 .png 图片，将作为水印全程覆盖)
  └── 📂 背景音乐/    (存放 .mp3, .aac 等音频文件)
```

### 2. 运行工具

#### 方式 A：直接运行源码

```
cd /path/to/SoupyClip
python3 video_processor_gui.py
```

#### 方式 B：双击打包好的 APP

直接双击 `dist/SoupyClip.app` 即可启动。

### 3. 开始混剪

1. 点击 **“📂 点击选择产品主目录”**，选中你的 `我的产品_HQ` 文件夹。
2. 观察路径是否自动填好。
3. 输入 **生成数量**（例如 50 条）。
4. 勾选 **“启用 AI 画面随机微调”**（强烈推荐）。
5. 点击底部的 **“🚀 开始执行批量混剪”**。
6. 去喝杯咖啡 ☕️，完成后文件夹会自动弹出。

## 🏗️ 开发与打包 (Development)

如果你修改了 `video_processor_gui.py` 源码，想重新打包成可执行文件：

1. 进入项目目录：

   ```
   cd /你的项目路径
   ```

2. 清理旧文件：

   ```
   rm -rf build dist
   ```

3. 执行打包命令：

   ```
   pyinstaller --onedir --noconsole --name "SoupyClip" video_processor_gui.py
   ```

4. 成品位于 `dist/SoupyClip.app`。

## ⚠️ 注意事项

- **素材要求**：视频素材片段建议**大于 1 秒**。如果素材太短，可能会因无法满足拼接时长而跳过。
- **权限问题**：首次运行 APP 如果提示“无法验证开发者”，请前往 `系统设置 > 隐私与安全性` 中点击 `仍要打开`。
